# 🛒 Evara - Stationery E-Commerce Website

A complete e-commerce platform for stationery products built with PHP and MySQL.

## ✨ Features

- 🛍️ **Product Management** - Add, edit, delete, and browse stationery products
- 🛒 **Shopping Cart** - Add/remove items, update quantities
- 👤 **User Authentication** - Register, login, and profile management
- 📦 **Order Processing** - Place orders and track history
- 🔍 **Search & Filter** - Find products by category, price, or keywords
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **Backend:** PHP
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript
- **Server:** Apache (XAMPP/WAMP)

## 🚀 Installation

1. Clone the repository
2. Set up XAMPP/WAMP
3. Import database
4. Update database config in `db.php`
5. Run the project
